# Файловое хранилище   
Реализованно на основе протокола http

## Сервер
Сервер является веб-приложением и для авторизации использует psql. Так же с ним приложен docker-compose файл, который иниуиализирует два контенйера: один для база данных, другой для самого сервер.   
Ограничений по файлам нету. Все файлы будут лежать рядом с исполняемым файлом в папке storage
## Клиент
Клиентская часть реализованно на основе avalonia.mvvm
